# Sysinfo Dashboard

Una herramienta ligera y eficiente para monitorear los recursos de tu sistema directamente desde la terminal, optimizada para entornos Linux y Termux.

## 📊 ¿Qué es esto?
Este dashboard proporciona una visualización rápida de datos críticos como el uso de **CPU**, **memoria RAM**, estado del **almacenamiento** y detalles del sistema operativo. Es ideal para mantener un ojo en el rendimiento de tu servidor o dispositivo móvil sin consumir recursos excesivos.

## 🚀 Instalación y Puesta en Marcha
Copia y pega los siguientes comandos en tu terminal (Termux o Linux):

### 1. Clonar el repositorio
```bash
git clone (https://github.com/george0884/sysinfo-dashboard)
```
Dar permisos de ejecución
Navega a la carpeta y otorga los permisos necesarios al script:

```cd sysinfo-dashboard```

```chmod +x sysinfo.sh```

Inicia el monitor con el siguiente comando:
```./sysinfo.sh```

📋 Requisitos Previos
Tener instalado Git (pkg install git en Termux o sudo apt install git en Ubuntu).
Entorno de terminal Bash.
