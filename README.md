# Sysinfo Dashboard

Una herramienta ligera para monitorear los recursos de tu sistema (CPU, RAM, Almacenamiento) directamente desde la terminal. Optimizado para Linux y Termux.

## 🚀 Instalación y Uso rápido

Copia y pega estos comandos en tu terminal:

### 1. Clonar el repositorio
```bash

git clone https://github.com/george0884/sysinfo-dashboard && cd sysinfo-dashboard && chmod +x sysinfo.sh && ./sysinfo.sh
